## Installation

### Backend
1. Navigate to `backend` folder
2. Create `.env` file based on `.env.example`
3. Run `npm install`
4. Run `npm start`

### Frontend
1. Navigate to `client` folder
2. Create `.env` file based on `.env.example`
3. Run `npm install`
4. Run `npm run dev`
